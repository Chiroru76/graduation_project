## サービス概要

日々の習慣やタスクの達成が、育てるの成長に直結します。
睡眠・勉強・運動などを記録するとステータスが変化し、タスクをこなすと経験値が貯まりレベルアップや進化が可能に。
一方で、習慣を続けなかったりログインを怠るとキャラは弱り、やがて「死」を迎えてしまいます。
ゲーム感覚で習慣形成や継続、タスクの管理をすることができるサービスです。

## このサービスへの思い・作りたい理由

私はランニングや勉強など日々の習慣として行なっていることのログを記録するようにしています。
それは、記録として残すことで、数値として自分の行動や積み重ねてきた努力が「可視化」されることで、
達成感や自己効力感を感じ、それが習慣を継続するモチベーションの一つになっているからです。

また、このような日々の努力の積み重ねや行動は、ゲームで主人公が経験値をゲットし成長していく
様子と重なる感覚があるため、楽しみながら続けることができていると感じています。

こうした経験から日々の行動がの結果が直接ペットの成長・進化という形で可視化できるゲーミフィケーション型のアプリを作りたいと考えました。

## ユーザー層について

- **「たまごっち」をはじめとする育成ゲームに馴染みのある方**
  ペットの成長やコレクション要素に懐かしさや楽しさを感じてもらいやすい。
  ゲーム感覚で習慣形成やタスク管理を取り入れることができる

- **今時点で何か毎日の習慣として続けていることがある、または新たに身につけたい習慣がある方**
  ペットの育成を、習慣を続ける理由の一つとして加えることで、違う面でのモチベーションが生まれ
  行動が継続しやすくなる。

- **味気ないタスク管理に物足りなさを感じる人**
  すでにメモアプリや TODO アプリを使っているに対し、完了したタスクを削除する又はチェックをする
  達成感に加えて、タスクの完了をペットのレベルアップ、進化に結びつけることで達成感が強化され、
  単なるタスク管理が楽しい体験に変わる。

## サービスの利用イメージ

### 【利用イメージ】

- 新規登録時に「たまご」を入手。初めてのレベルアップ時にペットが誕生する。
- ユーザーは日々の習慣（例：勉強、運動、読書等）や TODO を登録する。
- タスクを完了すると、ペットが経験値を獲得してレベルアップ、一定条件を満たすと進化する。
- 一方で、一定期間ペットの育成を怠るペットが死亡してしまうが、新しい「たまご」から育成を再スタートできる。
- 図鑑機能があり、自分が育てたペットの記録の振り返ることができる。

### 【得られる価値】

- **ゲーム感覚で自己管理**
  　自らの習慣の見直しや継続、タスクの管理を「ペット育成」という形に置き換えることで、
  　自然に楽しみながら行うことができる。

- **フィードバックによるモチベーションの向上**
  　自分の行動がペットの成長・進化あるいは死に直結するため、行動のモチベーションが生まれる。
  　また、習慣とTODOの達成数や行動量がグラフやカレンダーを通して可視化されることで、
  　達成感や自己肯定感を高めることができる。


## サービスの差別化ポイント・推しポイント

本アプリは [Habitica](https://habitica.com/static/home) と類似していますが、差別化ポイントは以下の通りです。

1. **多様なペット育成**
   Habitica はアバターのレベルアップによる着せ替えが中心ですが、本アプリでは「複数種類のペット」を育成可能。
   ユーザーごとに「推しペット」を見つけられるため、マンネリ化を防ぎ、継続利用を促す。

2. **ライフサイクル要素（誕生 → 成長 → 死）**
   ペットは成長するだけでなく、一定期間習慣を怠ると「弱り」「死」を迎える。
   この仕組みにより「やらなければ」という自然な動機づけが働き、習慣継続を後押する。

3. **リセットと再挑戦の楽しみ**
   ペットの死亡後は新しい「たまご」から育成を再スタートできるため、単調にならず、継続的に新しい体験が得られる。

4. **インタラクティブな体験**
   ペットはアニメーションで常に動き、条件に応じてコメントやリアクションを返す。
   単なる数字やゲージ管理にとどまらず、「キャラとの交流」を通じた愛着形成が可能。

#  - 機能一覧

---

## 🔐 ユーザー管理・認証

### 🔑 認証機能
- **新規登録**
  - メールアドレスとパスワードでの新規登録
  - ソーシャルログイン機能（Google・LINE）
  - メールアドレス再設定機能
  - パスワード再設定機能
  - 登録時に自動で「たまご」を入手

---

## ✅ タスク管理機能

- **TODO/習慣の作成**
  - タイトル設定
  - 期限設定　※TODOのみ
  - 難易度選択（簡単/普通/難しい）
  - 繰り返しルール設定（曜日指定）※習慣のみ
  - 目標設定（数値・単位・期間）　※習慣のみ
  - 記録方法を選択　※習慣のみ
    - **チェックボックス型**: ✅で記録。毎日自動で前日の✅をつけた習慣の✅表示をリセット（深夜0時）
    - **数値記録型**: 具体的な数値を記録（例: 30回、5km、2時間など）

---

## 🐾 ペット育成機能

### 📈 成長・進化機能
  - 新規登録時に「たまご」を入手
  - タスク完了時に獲得する経験値によるレベルアップ
  - タスクに設定した難易度に応じた経験値付与
  - 初回レベルアップ（Lv.1→Lv.2）で「たまご」からペットが誕生
  - Lv.9→Lv.10のレベルアップ時に進化
  - ペット画像のアニメーション表示(Webpの画像形式を採用)
  - ペットの誕生・進化のXシェア機能

### 🍖 えさやり機能
  - タスク完了時に入手できるエサをあげることで「しあわせ度」がUP

### 💀 ペットの死亡・リセット
  - 毎日自動で「しあわせ度」が減少（定期バッチ処理）
  - しあわせ度が0になるとペットが死亡
  - 新しい「たまご」の状態から育成を再スタート

### 💬 ペットコメント機能
  - OpenAI APIを使用したコメント動的生成
    - タスク完了時
    - えさやり時

### 🔔 LINE通知機能
以下の条件に当てはまるユーザーに対してLINE通知を行う
  - TODOの期限のリマインド通知
  - ペットの「しあわせ度」低下の警告通知
  - ペットが死んでしまった場合の通知

### 📊 グラフ表示機能
  - カレンダーにタスクの完了日をチェック
  - 直近7日/30日間のTODOと習慣の完了数を表示
  - 習慣（数値入力型）の実績値を表示

### 📖 図鑑機能
  - これまで育てたペットの一覧を表示

### 🏆 称号獲得機能
  - タスクの完了数,ペットのレベルに応じてユーザーに称号を付与
  - 獲得した称号はプロフィールページから確認可能
